#  House Price Prediction

##  Project Overview

This project focuses on predicting house prices using Machine Learning techniques. The objective is to analyze housing data, identify important factors affecting property prices, and build a predictive model capable of estimating house prices accurately.

The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, and performance evaluation.

---

##  Problem Statement

Real estate buyers and sellers often rely on guesswork or outdated comparisons to estimate a property's fair value. This project aims to develop a machine learning model that predicts house prices based on various property features.

---

##  Dataset

The dataset contains information about houses, including:

- Area
- Number of Bedrooms
- Number of Bathrooms
- Stories
- Parking Availability
- Furnishing Status
- Price (Target Variable)

##  Project Workflow

### 1. Data Loading
- Imported the housing dataset.
- Examined data structure and feature information.

### 2. Data Preprocessing
- Checked for missing values.
- Converted categorical variables into numerical format.
- Prepared data for machine learning.

### 3. Exploratory Data Analysis (EDA)
- Analyzed feature distributions.
- Studied relationships between variables.
- Identified key factors influencing house prices.

### 4. Data Visualization
Created visualizations to understand the dataset:

- Price Distribution
- Correlation Heatmap
- Actual vs Predicted Price Comparison

### 5. Model Building
- Split data into training and testing sets.
- Trained a Linear Regression model.
- Generated predictions on unseen data.

### 6. Model Evaluation
Evaluated model performance using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
